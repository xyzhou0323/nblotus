    脑脑启程丨一张车票，一段旅程，一个世界

    本文首发：脑脑空间NeuroBridge微信公众号
    首发时间：2025-04-22
    作者：Rossa

    内容概要：

    • 我们的理念定位
    • 我们的内容构成

    这是一份自我介绍。 link

    当你在浩瀚的网络信息中看到这些字句，当这些字句落入你的眼底，就像从书页间，被遗忘的车票飘落。
    幸运的是，这张车票永不过时。

    汽笛已然鸣响，月台人头攒动。
    我们翘首以盼，望眼迎接你的到来。
    旅程去往一个罕为人知、常被遗忘、屡遭误解的世界——属于神经殊异群体（neurodivergent，即ND）的世界。
    我们唤它作“脑脑空间”，并以此为名。
    “有些大脑喜欢跳舞，不走寻常路。
    这些大脑的“语言”不同，思考世界的方式和大多数人不一样，但同样精彩。

    这就是神经殊异群体（Neurodivergent，也译作“神经多样性群体”“神经多元群体”“神经发散群体”等） —— 包括ASD群体、ADHD群体、读写障碍群体等。”

    这是一个怎样的世界？
    对ND群体而言，这是素昧平生的故乡。
    对所有人而言，这个世界不卑不亢，真切诠释着“参差百态乃幸福之源”。

    为什么我们希望与你同行，搭起桥，一起去往这个世界？
    只因相信，神经多样性与每个人有关。
    那些被否定、被排斥的记忆，那些困惑、无助、沮丧的经历，在这个世界里被真正理解，获得力量。
    为了打破沉默的黑夜，我们要在这个世界发出自己的声音。
    为了ND群体，为了所有人。

    这是一段怎样的旅程？ link

    “脑脑空间”想把这些献给你：

    • 通俗的大众科普。 只有帮助人认识自己、理解自己、克服困境，知识才成为武器，而非枷锁。
      我们想传递关乎nd生活的、去污名化的知识，包括但不限于脑科学、心理健康、自助方法、社科理论。

    • 真诚的生命体验。 “理论是灰色的，而生活之树常青。”我们想讲述自己的故事，给思考以生命的温度，并期待与你共振。
      包括但不限于专业人士访谈、集体写作或赛博茶话、社会观察与思考分享。

    • 友善的交流空间。 孤独让人弱小懦弱。在与同类的沟通交流中，我们寻找到他人，寻找到自己，寻找到前进的信心与希望。
      我们想撑起一片友爱交流、共同讨论的净土。

    • 炽热的自我倡导。 “任何命运，不论如何漫长复杂，实际上只反映在人们大彻大悟自己究竟是谁的瞬间。”
      自我倡导是发声，为自己发声，为所有相同经历的人发声，最终让世界转过头来。
      我们想发出声音，传递勇气，共享生存的信心与力量。

    我们共享着一个世界，我们散落于不同“世界”。

    🌏

    但我们都向往着更好的世界。

    在“脑脑空间”里，那些没有血缘的兄弟姐妹们，携起手来。
